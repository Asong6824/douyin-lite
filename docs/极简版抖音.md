# 极简版抖音
## 项目结构
```
douyin
├── configs
├── docs
├── global
├── internal
│   ├── middleware
│   ├── routers
│   ├── service
│   ├── dao
│   └── model
├── pkg
│   ├── logger
│   ├── errcode
│   └── app
├── storage
├── scripts
└── third_party
```
- configs：配置文件
- docs：文档集合
- global：全局变量
- internal：内部模块
  - middleware：HTTP 中间件
  - routers：路由相关逻辑处理
  - service：对业务逻辑的封装
  - dao：对数据访问对象的封装
  - model：对模型操作进行封装，只与实体产生关系
- pkg：项目相关的模块包
  - logger：日志标准化
  - errcode：错误码标准化
  - app：
- storage：项目生成的临时文件
- scripts：各类构建，安装，分析等操作的脚本
- third_party：第三方的资源工具
